# idaproject_test

## 1 Create virtual env
`python3 -m venv vevn`
## 2 Enter the virtual env
`source venv/bin/activate`
## 3 Install Requirement
`pip install -r requirements.txt`
## 4 Install migrations
`./manage.py migrate`
## 5 Run Django server
`./manage.py runserver 8000`
